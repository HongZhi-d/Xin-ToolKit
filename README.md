### 上手指南

###### 开发前的配置要求

1. Visual Studio Community 2022
2. .NET 8.0

###### **安装步骤**

1. You don't need to get any keys to use it
2. Clone the repo

```sh
git clone https://github.com/HongZhi-d/Xin-ToolKit.git
```

### 开发的架构 

该项目架构基于 Windows x64 以及 Windows x86
